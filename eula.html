Declaração de Privacidade - SpotHub

Bem-vindo ao SpotHub. Esta Declaração de Privacidade estabelece como o SpotHub protege, armazena e trata as informações dos usuários que utilizam nossa plataforma.

1. Sobre o SpotHub

O SpotHub integra agências de publicidade e veículos de mídia, como emissoras de rádio e TV, facilitando o processo de envio, recebimento e tratamento de Pedidos de Inserção (PIs) e materiais publicitários por meio de inteligência artificial.

1.1 Tipos de Usuários

Existem três tipos de usuários na plataforma SpotHub:

• Agência: responsável pelo envio de PIs e materiais publicitários.

• Veículo: emissora de rádio ou TV que recebe o PI e processa as informações para execução da veiculação.

• Integração: software de operações comerciais (OPEC) que coleta dados do SpotHub para integração com o veículo de mídia.

2. Coleta e Tratamento de Dados

O SpotHub processa dados como informações de cadastro, PIs, materiais publicitários e contatos entre agências e veículos. Os dados são tratados exclusivamente para facilitar o fluxo de trabalho entre as partes.

2.1 Finalidade do Tratamento

O SpotHub utiliza os dados para:

• Processar e gerenciar Pedidos de Inserção e materiais publicitários.

• Facilitar a comunicação entre agências e veículos de mídia.

• Garantir a segurança e integridade dos dados conforme esta Declaração de Privacidade.

O SpotHub não utiliza os dados para nenhuma outra finalidade além da prestação de serviços.

2.2 Acesso Restrito e Controle de Usuários

Somente os seguintes usuários têm acesso aos dados:

• Agência responsável: acesso aos PIs e materiais publicitários enviados.

• Veículo: acesso aos PIs e materiais designados para veiculação.

• Integração: acesso aos dados integrados no sistema comercial do veículo.

Cada agência e veículo pode qualificar os usuários autorizados a visualizar e editar informações, usando opções disponíveis no portal.

3. Segurança e Proteção de Dados

O SpotHub protege os dados de seus usuários contra acesso não autorizado, perda, alteração ou divulgação indevida, seguindo as práticas abaixo:

• Criptografia de dados: todos os dados são criptografados em trânsito e em repouso.

• Acesso Controlado: apenas usuários autenticados e autorizados têm acesso aos dados.

• Auditoria e Monitoramento: auditorias periódicas e monitoramento contínuo são realizados para assegurar a conformidade.

4. Armazenamento de Dados e Conformidade com a LGPD

O SpotHub cumpre a Lei Geral de Proteção de Dados Pessoais (LGPD), mantendo os dados armazenados de forma segura e em conformidade com a legislação vigente.

4.1 Direitos dos Titulares

Os usuários do SpotHub têm o direito de:

• Acessar e Retificar: visualizar e solicitar correções de informações.

• Revogar ou Solicitar Exclusão: solicitar a exclusão dos dados, respeitando os prazos legais de retenção.

5. Compartilhamento de Dados

O SpotHub não compartilha os dados dos usuários com terceiros fora do circuito de comunicação definido. O compartilhamento de dados ocorre apenas entre agências, veículos e licenciados, ou quando exigido por lei.

6. Atualizações da Declaração de Privacidade

O SpotHub se reserva o direito de atualizar esta Declaração de Privacidade periodicamente. Recomendamos que os usuários revisem esta página regularmente para estarem informados sobre eventuais mudanças.
